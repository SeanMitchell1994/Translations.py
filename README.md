# Dependencies
* Python3 or higher
* Numpy 1.17.1 or higher 
    * All this probably works on earlier versions, but it hasn't been tested
    
# Usage  
1. Import the module
2. Create a Trans object
3. Call the functions you want to use

## Available Functions
* rotate_2d
* rotate_x_3d
* rotate_y_3d
* rotate_z_3d
